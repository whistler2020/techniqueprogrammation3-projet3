A propos


Arriver a la fin d’un programme en développement web (front end), la réalisation d’un projet web en anguler nous a été propose dans le cadre d’un cours en programmation web avec le Framework Angular. Cette interface ci est la concrétisation de ce projet Angular dont ces caractéristiques vont être développées dans les paragraphes suivants.

Présentation de cette application.


Cette application, créée en Angular, permet de faire des réservations de voyages. Elle est reliée à une base de données MySQL qui donne la possibilité d’une gestion de données des reservations ou des informations relatives au besoin des utilisateurs ou des clients ; comme par exemple, la liste des réservations, la suppression d’une réservation, la modification etc.


Composition de l’application

  • - Accueil : une page d’accueil qui sert l’entrée présentative du programme contenant l’affichage de certaines informations sur la reservation.
  • - Mini forfait : une page permettant d’afficher la liste des reservations partielle.
  • - Forfait Complet : une option permettant d’afficher une liste complète des reservations de forfait vedette selon la destination.
  • - A propos : c’est une section reversée pour fournir des informations générales sur la description de cette application, de ses fonctions, de son utilisation etc.
  • - Administration : c’est une des options importantes qui permet de faire la gestion de données, des réservations.

Mode d’utilisation

Pour les spécialistes ou des programmeurs qui souhaitent utiliser cette application dans le cadre de leur recherche, il faut préalablement installer et configurer sur son ordinateur les requis suivants:

  • - Nodejs : installation de « nodejs » (la version la plus récente serait mieux)
  • - NPM : installation de « NPM »
  • - Angular material : installation de « Angular Matériel »
  • - AMPPs / WAMPPS : installation de Ampps ou wampps ou mampps pour l’utilisation de MYQLS et d'autres modules encore selon le besoin.
